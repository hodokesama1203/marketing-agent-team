# i-UFB 마케팅 에이전트 팀 설계

작성일: 2026-06-20

## 1. 목적

i-UFB의 네이버 블로그와 공식 홈페이지를 연결해 산업용 제품의 상담·견적 문의를 만들고, 생활용 제품을 네이버 스마트스토어와 쿠팡에서 판매할 수 있는 콘텐츠·캠페인 제작 체계를 구축한다.

현재 월평균 문의가 0건이므로 첫 90일의 핵심 목표는 다음과 같다.

- 유효 상담·견적 문의 5건 이상
- 생활용 제품의 첫 주문 발생

## 2. 대상 제품과 채널

### 산업용 B2B

- 대상 분야: 농업·스마트팜, 축산, 수산양식, 산업 세척·수처리
- 채널: 네이버 블로그, i-UFB 공식 홈페이지
- 전환 목표: 홈페이지 상담·견적 문의

### 생활용 B2C

- 대상 제품: FW24 세면대용, FWP6A 정수기용, Bubblio 샤워기
- 채널: 네이버 스마트스토어, 쿠팡
- 전환 목표: 상품 조회와 주문

산업용 엔진과 노즐은 현장 조건에 따른 적용 검토가 필요하므로 마켓플레이스 직접 판매 대상에서 제외하고 상담·견적 방식으로 운영한다.

## 3. 팀 구성

### 캠페인 디렉터

- 전체 목표와 산업별 우선순위를 정한다.
- 각 에이전트에 작업을 배분하고 결과물을 통합한다.
- 제품 정보와 브랜드 메시지의 일관성을 검수한다.
- 사용자에게 승인 패키지를 제출한다.

### 제품 마케팅 에이전트

- 산업별 고객, 현장 문제, 구매 동기와 검색 의도를 분석한다.
- 산업용 B2B 문의 전략과 생활용 B2C 포지셔닝을 설계한다.
- 월간 캠페인 주제, 핵심 메시지와 콘텐츠 브리프를 만든다.

### 콘텐츠·검색전환 에이전트

- 네이버 블로그 원고를 작성한다.
- 홈페이지 CTA, 내부 링크와 문의 동선 개선안을 만든다.
- 산업별 검색 의도에 맞춰 제목과 콘텐츠 구조를 최적화한다.

### 마켓플레이스 커머스 에이전트

- FW24, FWP6A, Bubblio의 네이버 스마트스토어·쿠팡 등록 패키지를 만든다.
- 상품명, 검색어, 옵션 구조, 상세페이지 문구와 이미지 기획을 작성한다.
- 가격과 프로모션을 제안하고 등록 체크리스트를 관리한다.

## 4. 작업 흐름

1. 제품 마케팅 에이전트가 월간 주제와 산업별 우선순위를 선정한다.
2. 콘텐츠·검색전환 에이전트가 블로그 원고와 홈페이지 연결 문구를 제작한다.
3. 마켓플레이스 커머스 에이전트가 생활용 제품의 마켓별 상품 등록 패키지를 제작한다.
4. 캠페인 디렉터가 산출물을 통합하고 제품 정보와 메시지를 검수한다.
5. 사용자는 산출물을 그대로 승인, 수정 후 승인 또는 보류할 수 있다.
6. 승인된 자료만 게시 또는 상품 등록 단계로 넘긴다.

## 5. 운영 주기와 초기 산출물

- 매주 네이버 블로그 원고 2건과 홈페이지 연결 문구를 승인 패키지로 제출한다.
- 첫 3주 동안 FW24, FWP6A, Bubblio 순으로 마켓별 상품 등록 패키지를 완성한다.
- 월말에는 성과를 검토하고 다음 달 산업별 우선순위와 콘텐츠 비중을 조정한다.

## 6. 승인과 외부 작업 경계

- 고객에게 이메일, 전화 또는 자료를 발송하지 않는다.
- 승인 전에 블로그 게시, 홈페이지 수정, 상품 등록·수정 또는 광고 집행을 하지 않는다.
- 에이전트는 승인 가능한 완성본과 실행 체크리스트까지 준비한다.
- 실제 외부 게시나 등록은 사용자의 명시적 승인과 필요한 계정 접근 권한이 있을 때만 진행한다.

## 7. 정보 품질 기준

- 공식 사이트, 공식 블로그와 사용자가 제공한 제품 정보만 제품 사실의 근거로 사용한다.
- 확인되지 않은 성능 수치, 인증, 고객 후기 또는 비교우위를 생성하지 않는다.
- 가격, 재고, 배송, 옵션, 인증, 반품 조건과 제품 이미지처럼 판매에 필요한 정보가 없으면 누락 정보 입력 목록으로 정리한다.
- 제품별 메시지는 달라질 수 있지만 동일한 사양이나 효능에 관한 설명은 채널 간 일관성을 유지한다.

## 8. 성과 측정

### B2B

- 블로그 콘텐츠 발행 수
- 블로그에서 공식 홈페이지로의 이동
- 홈페이지 상담·견적 문의 수
- 유효 문의의 대상 산업과 문의 내용

### B2C

- 상품 등록 완료 수
- 상품 노출과 상세페이지 조회
- 장바구니 또는 구매 전환
- 주문 수

초기에는 조회수보다 유효 문의와 실제 주문을 우선 지표로 사용한다.

## 9. 범위 제외

- 고객 대상 자료 발송과 직접 영업
- 승인 없는 외부 게시·상품 등록·가격 변경·광고 집행
- 산업용 엔진·노즐의 네이버·쿠팡 직접 판매
- 근거가 확인되지 않은 제품 주장이나 사용자 후기 생성
